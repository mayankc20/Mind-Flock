# MIND-FLOCK

Code accompanying the paper

"Identity-Driven Relational Flocking with Dynamical Perceptual Coupling"

Authors:
Mayank Chaurasiya, Laxmidhar Behera

This repository contains the Python implementation used to generate the simulations and figures reported in the manuscript.

Requirements:
- numpy
- scipy
- matplotlib

Run the Jupyter notebook `MIND_FLOCK_2group_comarison_with_vicsek_and_similarty.ipynb` to reproduce the results.
